    给出一个字符串，找出有多少个子串满足重新排列后是一个回文串。

    Problem D. Exact Measurement

    Solved By Dup4 & groggy_. 04:35(+1)

    题意

    给出 个箱子，每个箱子有 的砝码有 个，现在要取最少的箱子，使得有一种方案取出这些箱子中的砝码使得重量和等于 。 即箱子中的砝码可以不全取出来。

    思路

    低位的砝码不能满足更低位的需求，但是能够满足高位的需求。 那么我们从地位到高位贪心，每次遍历到一位，都会解锁若干箱子，然后丢进最大堆，每次取盒子砝码总重量大的即可。

    Problem K. Knockout Racing

    Solved By groggy_. 00:25(+1)

    题意

    每辆车在 和 之间来回移动。 求 时刻， 和 区间内有多少辆车。

    思路

    计算查询时刻的每个车位置

    Code 
    #include <iostream>

#define maxn 1005
using namespace std;

int n, m;
int a[maxn], b[maxn];

int calc(int index, int t) {
    t %= 2 * (b[index] - a[index]);

    if (t <= b[index] - a[index])
        return a[index] + t;
    else
        return 2 * b[index] - a[index] - t;
}

void solve(int x, int y, int t) {
    int cnt = 0;
    for (int i = 0; i < n; i++) {
        int dis = calc(i, t);
        //		cout<<dis<<endl;
        if (dis >= x && dis <= y)
            cnt++;
    }
    printf("%d\n", cnt);
}

int main() {
    scanf("%d%d", &n, &m);
    for (int i = 0; i < n; i++) {
        scanf("%d%d", &a[i], &b[i]);
    }

    int x, y, t;
    for (int i = 0; i < m; i++) {
        scanf("%d%d%d", &x, &y, &t);
        solve(x, y, t);
    }
}
